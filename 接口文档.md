# Authentication

# 用户相关接口

## POST 用户登录接口

POST /users/login

> Body 请求参数

```json
{
  "username": "super_admin",
  "password": "123456"
}
```

### 请求参数

|名称|位置|类型|必选|说明|
|---|---|---|---|---|
|Authorization|header|string| 否 |none|
|body|body|[LoginFormDTO](#schemaloginformdto)| 否 |none|

> 返回示例

> 200 Response

```json
{
  "code": 0,
  "msg": "",
  "data": {
    "token": "",
    "userId": 0,
    "username": ""
  }
}
```

### 返回结果

|状态码|状态码含义|说明|数据模型|
|---|---|---|---|
|200|[OK](https://tools.ietf.org/html/rfc7231#section-6.3.1)|none|[ResultUserLoginVO](#schemaresultuserloginvo)|
